# The Chinese characters in this file using "UTF-8" code.

本目录中针对Lab3-1的实验任务要求给出了一个五级流水CPU的框架代码，希望能够同学们提供一些参考。

我的编码风格与我在piazza上发的实验提示是一套思路出来，其最大的特点就是每一级流水是从流水线寄存器开始到下一级流水线寄存器的D输入端之前。

每一级流水线寄存器我都加了前缀，通过阅读代码可以看出来。

我给的框架代码中，声明reg的一定需要实现为register。

用于控制信息的op的位宽我没有定死，目前在代码里都是??的形式，你们根据自己的理解修改。同时这也意味这这份代码完全不做修改，放到vivado里面是编不过的。
